## Django app on python

### Step 1

Copy github proejct

```bash
git@github.com:Pro7it/Django_lab3_python.git
cd Django_lab3_python
```

### Step 2

Contact us to add your key to our proejct

### Step 3

Make migrations

```
python3 manage.py makemigrations
python3 manage.py migrate main
```

### Step 4

Run app!

```bash
python3 manage.py runserver
```
